# Specification

## Summary
**Goal:** Add admin login links to the header and homepage for easy access to the admin dashboard.

**Planned changes:**
- Add an admin login link to the Layout.tsx header component that navigates to /admin
- Add an admin login shortcut link to the HomePage.tsx hero section or below it

**User-visible outcome:** Users will see admin login links in the header and on the homepage that navigate to the admin dashboard when clicked.
